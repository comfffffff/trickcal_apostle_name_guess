# 트릭컬 사도 이름 외우기 🐾

트릭컬 리바이브의 사도 이름을 외워서 입력하는 타임어택 웹게임입니다.
순수 HTML/CSS/JavaScript로 만든 정적 사이트라 별도 설치 없이 `index.html`만 열면 바로 플레이할 수 있습니다.

## 플레이 방법
- 사도들의 이름을 전부 입력하면 완료!
- 첫 화면에서 두 옵션을 조합해 시작합니다.
  - **가나다순으로 입력**: 켜면 가나다 순서대로, 끄면 자유 순서
  - **이격 사도 포함**: 켜면 이격(스킨)까지 포함, 끄면 기본 사도만
- 막힐 땐 **힌트**를 쓸 수 있고(다음/미입력 사도의 첫 글자), 결과창에서 소요 시간과 힌트 사용 횟수를 보여줍니다.

## 폴더 구성
- `index.html` · `style.css` · `script.js` — 화면/디자인/게임 로직
- `data.js` — 사도 데이터(`APOSTLES`)와 명단 업데이트 일자(`LAST_UPDATED`)
- `images/` — 사도 일러스트
- `fonts/` — 폰트(ONE Mobile POP)
- `assets/` — 배경 이미지
- `apostles.json` — 사도 데이터 참고용 사본(게임은 `data.js`만 사용)

## 저작권
본 프로젝트는 공식이 아닌 **비영리 팬메이드** 콘텐츠이며,
사용된 게임 리소스(이름·일러스트 등)의 저작권은 원저작자(에피드게임즈)에게 있습니다.
